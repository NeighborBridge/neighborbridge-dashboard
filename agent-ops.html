🛡️ Approval Levels / 审批等级
Level 名称 Agent 可做 Richard 必须批准？ 示例
0 Internal note only 生成内部 note / dashboard suggestion ❌ 无需发送 Dashboard status update
1 Draft only 起草 email / message / calendar / concept note ✅ 必须审核 Holly draft email
2 Send after approval 在明确批准后发送或提交 ✅ 批准后发送 Cindy / Vinod outreach
3 Limited auto-send 🚫 低风险白名单固定模板 ✅ 默认禁用 "Thank you, received"

Current default: All external communication is Level 1 or Level 2 only. Level 3 disabled.

当前默认：所有外部沟通均为 draft-only 或 Richard 批准后发送；不启用全自动发送。

  • • Search Gmail for known stakeholders
  • • Summarize new relevant messages
  • • Update dashboard status fields
  • • Draft replies · Place draft in approval queue

❌ Prohibited:

  • • Automatic external email sending
  • • Mass outreach · Sending funding asks
