## 2. Mapped Additions

### PRIORITY A (Map first — direct caregiver pain points, low risk)

#### A1. Caregiver Observation Log
| Field | Value |
|---|---|
| **Fits after** | Between Page 2 (Behavior as Communication) and Page 3 (Brain-Based Expectations) |
| **Format** | Worksheet (single page) — front of booklet insert, not a page number |
| **Why useful** | Bridges "look for patterns" (Page 2 skill) into a concrete tool. Old Trillium SOP validated standardized question set as essential. Caregivers need a non-clinical way to describe behavior to themselves before describing it to others. |
| **Risk level** | 🟢 LOW — purely descriptive, no diagnosis, no referral |
| **Clara trigger** | 🟢 None — caregiver self-tracking, not clinical data |
| **Priority** | **A** |

**Suggested structure**: 3 columns — Before (time/context) → During (behavior description) → After (what helped). Plus: "What might my child have been trying to tell me?" (connects to Page 2).

#### A2. Meltdown Debrief Card
| Field | Value |
|---|---|
| **Fits after** | Page 4 (Sensory Overload and Shutdown) or Page 5 (Co-Regulation) |
| **Format** | Card (separate, pocket-sized) |
| **Why useful** | Old sample's Red Flags + What-to-Do pattern is directly relevant. After-meltdown reflection helps caregivers shift from reactive frustration to pattern recognition. Low-cost, high-impact. |
| **Risk level** | 🟢 LOW — reflective, non-diagnostic |
| **Clara trigger** | 🟢 None |
| **Priority** | **A** |

**Structure**: "After a hard moment, take 3 deep breaths. Then: What happened right before? What did I try? What worked? What didn't? What might I try next time?" — No blame, no analysis.

#### A3. PCP Discussion Question Sheet
| Field | Value |
|---|---|
| **Fits after** | Page 9 (When to Seek Professional Help) |
| **Format** | Worksheet (1 page, detachable) — appendix or back-cover pullout |
| **Why useful** | Directly supports NeighborBridge's connector support platform role. Helps families prepare for PCP/medical home discussion without creating referral expectation. Old Trillium SOP's #1 caregiver concern ("Why does my child act this way?") is exactly what this sheet helps families articulate. |
| **Risk level** | 🟡 MEDIUM — must not imply PCP discussion guarantees evaluation or referral. Must be framed as "questions to help you describe what you're seeing" not "questions to get what you want." |
| **Clara trigger** | 🟡 Clara must review for referral pathway implication and access expectation |
| **Priority** | **A** |

**Suggested structure**: 5–7 questions like "What behaviors concern me most?", "When did I first notice?", "What makes it better or worse?", "What have I already tried?" — framed as preparation, not request.

#### A4. Sleep-Behavior Page
| Field | Value |
|---|---|
| **Fits after** | Page 4 (Sensory Overload) — or as Page 4a |
| **Format** | Page (expand existing Page 4 or separate sub-page) |
| **Why useful** | Old Trillium SOP identified caregiver exhaustion as Core Concern 5. Sleep-behavior connection is a known caregiver pain point but current booklet does not address it. Simple non-clinical practical tips. |
| **Risk level** | 🟢 LOW — practical, non-clinical |
| **Clara trigger** | 🟢 None — no sleep protocol, no medical advice |
| **Priority** | **A** |

**Structure**: Sleep challenges as sensory/regulation issue. Practical: bedtime routine ideas, winding down, reducing stimulation before bed. Explicit boundary: "This is not medical sleep advice. Talk to your child's doctor about persistent sleep concerns."

#### A5. School-Home Pattern Comparison
| Field | Value |
|---|---|
| **Fits as** | Expansion of existing Page 7 (currently lists school-home as a skill section) |
| **Format** | Add to Page 7 content or create Page 7a as worksheet |
| **Why useful** | Old Trillium SOP listed "school problems / misunderstood as ADHD/ODD" as Core Concern 3. Page 7 currently describes school-home differences but has no tool for capturing them. A simple comparison worksheet turns description into actionable information. |
| **Risk level** | 🟢 LOW — descriptive, collaborative |
| **Clara trigger** | 🟢 None |
| **Priority** | **A** |

**Structure**: Side-by-side columns: "At school my child…" / "At home my child…" + "What works in each setting?" — Helps families see pattern, not contradiction.
