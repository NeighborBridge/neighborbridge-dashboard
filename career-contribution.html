职业定位与学术贡献

从北卡农村基层医疗经验出发，提炼 global mental health、primary care、医学人类学与实施科学之间的共同主线

🔥 当前核心行动问题

如何把 Eastern North Carolina rural FQHC 的经验，提炼成一个 global mental health + primary care + FASD/NDD early-identification pathway 的主场域叙事？

引导问题：

  1. 北卡农村经验到底贡献了什么？
  2. 它如何区别于一般 FASD 项目、一般 rural health 项目、或一般 primary care integration 项目？
  3. 它如何成为未来 multi-site 比较的 primary site？
💡 当前工作答案

一、北卡农村经验贡献了什么

  • • 它把 global mental health 的 task-sharing / community-based care / primary care integration 转化到美国农村 Medicaid / FQHC 场景中，证明了这些模式在 FQHC 制度框架下的可行性
  • • 它揭示儿童 FASD / NDD / behavioral health 的核心问题不是单一诊断，而是早期识别、家庭解释、学校反馈、基层医疗、精神科转诊和持续支持之间的系统性断裂
  • • 它把 FQHC 变成 primary-care anchored early-identification site：基层医疗既是入口，也是协调中枢
  • • 它把 payer alignment / Medicaid MCO / high-cost utilization 作为美国场景中特别重要的实施变量——路径必须有 payer-facing value logic 才能持续

二、它与一般项目的不同

不是一般 FASD awareness 项目

不是一般 rural health access 项目

不是一般 primary care integration 项目

而是一个以 FASD/NDD 为入口、以基层医疗为锚点、以家庭-学校-医疗-社区衔接为核心、以 payer-facing value logic 为美国场景变量的 relational pathway-building model。

三、它为什么能成为 primary site

  • • 北卡农村 FQHC 是已有真实临床经验和本地关系的主场域
  • • 它同时包含临床、家庭、学校、Medicaid payer、FQHC workflow 和社区资源断裂的全部环节
  • • 它可以作为未来比较的基准场域——St. Louis、Appalachia、Mississippi Delta、湘西都不是替代北卡，而是与北卡形成对照
  • • 北卡提供 primary case，其他 site 提供 comparison cases

当前结论：

North Carolina rural FQHC should be treated as the primary field site and narrative anchor; other sites should be added later only as comparison sites.

北卡农村 FQHC 应作为主场域和叙事锚点；其他地点之后只作为比较场域逐步加入。

我的工作位于全球精神卫生、基层医疗、医学人类学和实施科学的交界处。核心关注是在资源不足的农村和安全网医疗场景中，如何建立儿童神经发育与行为健康问题的早期识别和连续支持路径，并解释这些路径如何通过信任、家庭、社区关系和基层流程真正运行起来。

🌟 核心学术贡献

Relational pathway-building in primary care

把 global mental health 从 service scale-up / task-sharing 推进到 基层医疗中的关系性路径建设

🌾 北卡农村作为主场域
▸ FQHC — 美国农村低资源场景中的 primary care 实验场
▸ Medicaid / MCO — payer alignment 成为美国全球精神卫生实践的重要变量
▸ FASD / NDD — 儿童行为健康与神经发育服务断裂的高价值切入点
▸ 家庭 / 学校 / 基层医疗 / 精神科 / 社区资源之间的断裂 — pathway-building 的核心问题
▸ CHW / family navigation — 把早期识别转化为持续支持的关键机制
🧬 人类学层

制度设计不会自动产生连续照护。

Implementation becomes possible through relational care.

  • • 信任、家庭、地方关系、道德责任和临床判断，是服务路径真正运行的条件
  • • 湘西 manuscript 与北卡农村经验可以形成理论对话：不同制度下，基层医疗都必须通过地方关系和照护实践才能运行
🔬 实施科学层

不是复制固定模型，而是研究 early identification pathways 如何在不同低资源场景中被本地化、适配和维持。

关注 pathway adaptation，而不仅是 model fidelity。

比较维度：

  • • 谁是入口？
  • • 谁建立信任？
  • • 断裂在哪里？
  • • 什么让路径持续？
  • • 什么最容易失败？
