北卡农村 FQHC 的核心价值

北卡农村 FQHC 的价值，不只在于"这里有一个 FASD/NDD 项目"，而在于它展示了 Richard 如何把 global mental health 的 task-sharing、community-based care、primary care integration 和 implementation thinking，转译到美国农村 Medicaid 场景中。

同一战略的不同执行层

🧪 CHW / family navigator dashboard — strategy execution tool，不只是产品
💰 Trillium / Medicaid MCO — payer-facing implementation layer，不只是 funding source
🏛️ Holly / county public health / school nurses / WIC / child health clinics — public health entry-point layer
🏥 FQHC — clinical anchor
🎓 Yale / Harvard / Duke — academic legitimacy and field-contribution layer

一、北卡农村经验贡献了什么

  • • 它把 global mental health 的 task-sharing / community-based care / primary care integration 转化到美国农村 Medicaid / FQHC 场景中，证明了这些模式在 FQHC 制度框架下的可行性
  • • 它揭示儿童 FASD / NDD / behavioral health 的核心问题不是单一诊断，而是早期识别、家庭解释、学校反馈、基层医疗、精神科转诊和持续支持之间的系统性断裂
  • • 它把 FQHC 变成 primary-care anchored early-identification site：基层医疗既是入口，也是协调中枢
  • • 它把 payer alignment / Medicaid MCO / high-cost utilization 作为美国场景中特别重要的实施变量——路径必须有 payer-facing value logic 才能持续

二、北卡为什么能成为 primary site

  • • 北卡农村 FQHC 是已有真实临床经验和本地关系的主场域
  • • 它同时包含 clinical care、family burden、school feedback、Medicaid payer、public health entry points 和 community fragmentation 的全部环节
  • • 它适合成为未来 multi-site 比较的 reference site
  • • 其他地点不是替代北卡，而是与北卡形成比较

当前结论：

North Carolina rural FQHC should be treated as the primary field site and narrative anchor; other sites should be added later only as comparison / expansion sites.

北卡农村 FQHC 应作为主场域和叙事锚点；其他地点之后只作为比较或扩展场域逐步加入。

🧬 FASD/NDD 的角色 clinical entry point

FASD/NDD 不是 Richard 全部事业的边界，而是当前最有战略价值的 clinical entry point。

它之所以重要，是因为它天然跨越 primary care、behavioral health、schools、family support、public health 和 Medicaid cost concerns，能够把儿童行为健康与神经发育服务断裂清楚呈现出来。

核心学术贡献：Relational pathway-building in primary care — 把 global mental health 从 service scale-up / task-sharing 推进到基层医疗中的关系性路径建设。

🎓 Yale / Harvard / Duke 应该看到什么 学术视角

这些学术机构不应只看到一个地方 FASD 项目，而应看到 Richard 的全局能力：

  • • 将地方田野经验转化为可比较、可推广、可理论化的 pathway-building framework
