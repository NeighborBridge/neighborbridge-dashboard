## Decision Logic for Choosing Among Alternatives

When the time comes to choose a path (not now), these criteria should drive the decision:

1. **What the work is** — Does the institution value community-engaged implementation work as a scholarly identity?
2. **Political cover** — Is there a department chair or senior leader who understands FASD/NDD as a real health system problem?
3. **Protected time** — Can the role offer ≥50% non-clinical time for pathway building?
4. **Local relevance** — Is the institution in a state where Medicaid transformation / value-based care creates an active need for this work?
5. **Income viability** — Does the institution support soft-funded or grant-funded implementation roles?
